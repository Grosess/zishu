# Zishu (字书)

**Master Chinese character writing with authentic stroke order guidance**

Zishu is a Flutter mobile application designed to help learners master Chinese character writing through interactive practice with authentic stroke order guidance powered by the MakeMeAHanzi database.

## Features

- ✍️ **Stroke-by-stroke practice** - Learn proper stroke order for thousands of Chinese characters
- 📚 **Pre-built character sets** - HSK levels, radicals, and more
- 🎯 **Custom practice sets** - Create your own character lists
- 📊 **Progress tracking** - Monitor your learning with streaks and statistics
- 🗣️ **Audio pronunciation** - Hear native pronunciation using text-to-speech
- 📖 **Dictionary integration** - View definitions and example words from CC-CEDICT
- 🌍 **Multi-language support** - Interface available in English and Chinese
- 🎨 **Customizable practice** - Adjust brush styles, hint colors, and more

## Download

- **iOS**: Available on the [App Store](https://apps.apple.com/app/zishu/id6738848149)
- **Android**: Coming soon

## Technology

Built with Flutter and Dart, using authentic Chinese character data from:
- **MakeMeAHanzi** - Stroke order and character structure data
- **CC-CEDICT** - Chinese-English dictionary
- **Unicode Unihan Database** - Character information
