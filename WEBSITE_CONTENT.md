# Zishu - Master Chinese Character Writing

## App Overview

Zishu (字书) is a comprehensive Chinese character learning app that helps users master authentic Chinese handwriting through stroke-by-stroke practice with real-time validation.

## Key Features

### 🎯 Core Learning System

#### Stroke-by-Stroke Practice
- **Authentic Stroke Data**: Uses MakeMeAHanzi database for accurate stroke order and direction
- **Real-time Validation**: Instant feedback on stroke accuracy and order
- **Visual Hints**: Animated stroke hints after incorrect attempts
- **Progress Tracking**: Tracks practice count and success rate for each character

#### Learning Modes
1. **Learning Mode** (3 stages):
   - Stage 1: Full hints with animated stroke guidance
   - Stage 2: Character outline only
   - Stage 3: No visual aids - pure practice
   
2. **Testing Mode**: Practice without hints to test knowledge
3. **Endless Practice**: Spaced repetition system for learned characters

### 📚 Character Sets

#### Built-in Sets
- **HSK Levels 1-6**: Official Chinese proficiency test vocabulary
- **Radicals**: Fundamental character components
- **Common Characters**: Frequently used characters in daily life
- **Numbers**: Chinese numerical characters

#### Custom Sets
- Create personalized character lists
- Organize sets into folders
- Import characters from text
- Search and add from 9000+ character database

### 🎨 Customization Options

#### Themes
- Light Mode
- Dark Mode
- Duotone Themes: Minimalist two-color designs
  - Multiple color combinations available
  - Custom background and foreground colors

#### Stroke Styles
1. **Simple**: Clean, minimal strokes
2. **Classic**: Smooth calligraphy brush with dynamic tip color
3. **Dynamic**: Width varies with drawing speed
4. **Neon Glow**: Electric effect with glow
5. **Rainbow**: Colorful gradient strokes
6. **Invisible**: Practice without visual feedback

#### Visual Settings
- Adjustable stroke width
- Custom stroke colors
- Grid display toggle
- Character outline visibility
- Animated hints on/off

### 📊 Progress Tracking

#### Statistics
- Daily practice goals with visual progress
- Streak tracking with motivational messages
- Character-specific statistics (attempts, success rate, time)
- Session summaries

#### Learning Analytics
- Practice history with detailed logs
- Success rate tracking
- Time spent per character
- Mistake pattern analysis

### 🔄 Data Management

#### Import Options
1. **Text Import**: Paste any Chinese text to mark characters as learned
2. **Skritter Import**: Import your learned characters from Skritter exports (.tsv/.csv)
3. **Backup Import**: Restore from app backup files

#### Export Features
- Full data backup as JSON
- Character set export
- Cross-device sync via manual backup/restore

### 🎯 Smart Features

#### Radical Analysis
- Breaks down characters into component radicals
- Shows radical meanings and pronunciations
- Helps understand character composition

#### Character Information
- Multiple pronunciation sources (Pinyin)
- Character definitions and meanings
- Traditional/Simplified variants
- Usage examples

#### Adaptive Practice
- Focuses on frequently mistaken characters
- Adjustable daily goals (10-99,999 characters)
- Smart review scheduling

### 🏃 Practice Modes

#### Single Character Practice
- Focus on one character at a time
- Repeat practice with counter
- Immediate feedback

#### Word Practice
- Practice multi-character words
- Character-by-character progression
- Maintains context while learning

#### Set Practice
- Work through entire character sets
- Progress tracking per set
- Visual progress bars
